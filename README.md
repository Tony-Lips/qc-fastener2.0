# 🔩 ОПМ — Контроль качества крепежа

PWA-приложение для контроля качества крепежа по ГОСТ ISO 3269-2021.

## ✨ Функции

- 📷 **Сканирование актов** с камеры телефона
- 🤖 **OCR распознавание** номера, кода, товара, количества
- 📐 **Выборка по ISO 3269-2021** (категории 1, 2, 3)
- 📖 **Шпаргалка приёмщика** с категориями контроля
- 🗑️ **Удаление позиций**
- 📋 **Матрица несоответствий** с выбором действий по ГК РФ
- 📸 **Фото дефектов** (до нескольких на позицию)
- 📄 **Генерация DOCX акта** по шаблону ООО «ОПМ»
- 📧 **Рапорт руководителю** на avg@opm.ru
- 💾 **История** в локальном хранилище
- 📲 **PWA** — устанавливается на телефон
